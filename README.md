# 📄 Job Application Form

A simple and modern Job Application Form built with pure HTML & CSS.
This project demonstrates clean form structure, semantic HTML, basic styling and button animations.

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (including transitions)

---

## 📂 Project Structure

```
project/
│── index.html
│── style.css
│── README.md
```

---

## 📸 Preview

The form includes:

- Name input field
- Email input field
- Position selector
- Availability (Full-Time / Part-Time)
- Motivation textarea
- Submit button

---

## ▶️ How to Run 1. Clone or download this repository 2. Open the project folder 3. Open index.html in your browser

---

## 💡 Possible Improvements

- Add form validation with JavaScript
- Connect form to a backend (e.g. PHP, Node.js)
- Improve accessibility (ARIA labels, better focus states)
- Add success/error messages after submission
- Enhance responsiveness for mobile devices

---

## 📌 Notes

- The form currently does not submit data to a server (no backend integration)
- All validation is handled via built-in HTML features
